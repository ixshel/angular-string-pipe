Strings Pipe



Option Result Description
string | string : "reverse" {{ example | string: 'reverse' }} Print reverse string
string | string : "firstCap" {{ example | string: 'firstCap' }} Print only the first letter of the string in capital letter
string | string : "capAllWords" {{ example | string: 'capAllWords' }} Print all words with capital letter
string | string : "noSpaces" {{ example | string: 'noSpaces' }} Remove all spaces
string | string : "truncate" : number {{ example | string: 'truncate' : 10 }} Truncate string after the number we gave in this example is 10
